Work Project

This project was made to test out not only coding, but also world building. I wanted to create a game, I already had an idea and an inspiration from the moving statues (weeping angels) from Doctor Who.


To make this game, I first explored some Official Roblox Studio Videos and Tutorials. I made a simple game with some GUI. This game was just a Test and to familiarize and to learn about the Studio. The world building aspect was particularly interesting and challenging. The coding was made by AI and revised by me. The coding wasnt hard, but the integration to make sure that every single part inside the game functioned correctly alongside other parts (to call on other objects and to be in the right container) was pretty hard.
